## How to Influence the Roadmap

1. **Vote** — React with 👍 on [existing feature requests](https://github.com/nirholas/pump-fun-sdk/issues?q=label%3Aenhancement)
2. **Propose** — Open a [feature request](https://github.com/nirholas/pump-fun-sdk/issues/new?template=feature_request.md)
3. **Discuss** — Start a [discussion thread](https://github.com/nirholas/pump-fun-sdk/discussions)
4. **Build** — Submit a PR — the best way to get a feature shipped is to build it
5. **Sponsor** — [Fund development](https://github.com/sponsors/nirholas) to accelerate specific features
