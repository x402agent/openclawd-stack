## 🐛 Found a Bug?

[Open a bug report](https://github.com/nirholas/pump-fun-sdk/issues/new?template=bug_report.md) with:

- Your environment (OS, Node.js version, SDK version)
- Steps to reproduce
- Expected vs. actual behavior
- Error messages or logs

---

## 💡 Have a Feature Idea?

[Open a feature request](https://github.com/nirholas/pump-fun-sdk/issues/new?template=feature_request.md). Describe:

- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

## Response Times

This is an open-source project maintained by volunteers. We do our best to respond quickly:

| Channel | Typical Response |
|---------|------------------|
| Security reports | Within 48 hours |
| Bug reports | Within 1 week |
| Feature requests | Within 2 weeks |
| Discussions | When we can |

Please be patient and respectful. We're building this for everyone.
