**PumpKit is live** — an open-source TypeScript framework for building production-ready PumpFun Telegram bots on Solana.

What's inside:
• `@pumpkit/core` — Bot scaffolding, event monitors, formatters, storage, health checks
• `@pumpkit/monitor` — All-in-one PumpFun monitor (claims, launches, graduations, whales, CTOs)
• `@pumpkit/tracker` — Group call-tracking bot with leaderboards, PNL cards, and rankings
• `@pumpkit/channel` — Read-only channel feed for token events
