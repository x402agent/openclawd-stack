## Built With

This project wouldn't be possible without these amazing open-source projects:

### Solana Ecosystem

- **[Solana Labs](https://github.com/solana-labs)** — The foundation: `solana-sdk`, `@solana/web3.js`, `solana-keygen`
- **[Coral (Anchor)](https://github.com/coral-xyz/anchor)** — Program framework and IDL tooling
- **[SPL Token](https://github.com/solana-labs/solana-program-library)** — Token program interface

### AI & Agent Infrastructure

- **[Anthropic (Claude)](https://anthropic.com)** — Model Context Protocol specification
- **[Model Context Protocol SDK](https://github.com/modelcontextprotocol/sdk)** — MCP server/client libraries
- **[SperaxOS](https://sperax.io)** — AI-powered DeFi operating system

### Rust Ecosystem

- **[Rayon](https://github.com/rayon-rs/rayon)** — Data parallelism for multi-threaded vanity generation
- **[Zeroize](https://github.com/RustCrypto/utils/tree/master/zeroize)** — Secure memory cleanup for private keys
- **[Clap](https://github.com/clap-rs/clap)** — Command-line argument parsing
- **[Serde](https://github.com/serde-rs/serde)** — Serialization framework

### TypeScript Ecosystem

- **[TypeScript](https://www.typescriptlang.org/)** — Type-safe JavaScript
- **[tsup](https://github.com/egoist/tsup)** — TypeScript bundler
- **[Jest](https://jestjs.io/)** — Testing framework
- **[ESLint](https://eslint.org/)** — Code linting
- **[bn.js](https://github.com/indutny/bn.js/)** — Big number arithmetic
